# flida_auth_sdk_ios

The iOS implementation of `flida_auth_sdk`.

## Usage

This package is endorsed, which means you can simply use `flida_auth_sdk` normally. This package will be automatically included in your app when you do.

### Configuration

Ensure you have added necessary keys to your `Info.plist`:

- `FlidaAuthHost`
- `CFBundleURLTypes`

See the main [flida_auth_sdk](https://pub.dev/packages/flida_auth_sdk) package for more details.
